Welcome to our official project space for the AI and Applications (ITE351) course at Hanyang University (Fall 2025).

The final project topic is currently under discussion and will be finalized after brainstorming and instructor feedback.

Project Proposal (Draft)

We plan to explore directions such as:
- Applying AI and ML techniques to analyze datasets and predict outcomes.
- Building a small interactive prototype integrating machine learning features.
- Demonstrating how AI simplifies real-world tasks through automation or intelligent recommendations.

We aim to create a reproducible workflow and document every step from data collection to evaluation.

Planned Methodology

  1. Topic Selection - identify a meaningful AI problem (healthcare, environment, automation, etc.)
  2. Data Collection and Preprocessing - find or create suitable datasets
  3. Modeling - train and test models using Python and ML frameworks
  4. Evaluation - analyze results and optimize performance
  5. Documentation - publish updates on this GitHub Pages blog
