Project Overview

This website contains the project workspace for Group 34 in the AI and Applications course (ITE351).
Our final project explores building a machine learning model that predicts whether a loan application
will be approved or rejected based on applicant financial, demographic, and credit-related features.
The dataset used is from Kaggle: Loan Approval Classification Data.

Project Proposal (Loan Approval Prediction)

The goal of this project is to build an AI-powered system that assists loan approval decisions by analyzing applicant information such as income, credit score, employment history, education level, and past default records. Financial institutions increasingly rely on data-driven systems to ensure fast, consistent, and fair evaluations.

Using a structured dataset of 45,000 applicants, we perform exploratory data analysis, feature engineering, and model training to understand which variables most strongly influence loan approval outcomes. Models such as Logistic Regression, Random Forest, and XGBoost will be compared to evaluate performance and interpretability.

Ultimately, this project aims to develop a reliable classification model and provide insight into how applicant characteristics influence approval decisions in real-world loan screening scenarios.

Planned Methodology

  1. Data Loading and Inspection
    Import dataset from Kaggle and examine entries, datatypes, missing values, and basic structure.

  2. Exploratory Data Analysis (EDA)
    Visualize numeric and categorical variables, study distributions, identify outliers, analyze target imbalance, and explore correlations.

  3. Preprocessing and Feature Engineering
    Handle outliers, encode categorical variables, scale numeric features, and prepare training and test datasets.

  4. Model Development
    Train and compare multiple machine learning models:
    • Logistic Regression
    • Random Forest Classifier
    • XGBoost Classifier
    Models will be evaluated based on accuracy, precision, recall, F1-score, and ROC-AUC.

  5. Model Evaluation and Interpretation
    Analyze feature importance, compare algorithms, and interpret key predictors such as credit score, loan percent of income, and past default history.

  6. Documentation and Deployment
    Summarize findings, maintain GitHub repository structure, and prepare final report and presentation.
