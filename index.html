Project Overview

This project provides an end-to-end machine learning pipeline to predict loan approval outcomes (approved = 1 / rejected = 0). It includes exploratory data analysis, reproducible preprocessing, model training (Logistic Regression, Random Forest, XGBoost), evaluation, and interpretability via SHAP. The repository also contains reproducible Jupyter notebooks and an academic-style report.

Key Features

  • Reproducible preprocessing: missing value handling, encoding, scaling (src/preprocessing.py).
  • Modeling: train and compare Logistic Regression, Random Forest, XGBoost (src/train_model.py).
  • Evaluation: accuracy, precision, recall, F1, ROC-AUC and visualizations (src/evaluate.py).
  • Interpretability: feature importance and SHAP analysis (src/feature_importance.py).
  • Notebooks: step-by-step EDA → preprocessing → modeling → evaluation (notebooks/).

Data

Raw dataset: data/raw/loan_data.csv

Processed artifacts: data/processed/cleaned_data.csv, X_train.npy, X_test.npy, y_train.npy, y_test.npy

See notebooks/02_preprocessing.ipynb for column descriptions and preprocessing steps.

Installation

  1. Create and activate a virtual environment 
    python3 -m venv venv
source venv/bin/activate
  2. Install dependencies 
    pip install -r requirements.txt
  3. (macOS) Optional for XGBoost 
    brew install libomp

Quick Start

Run the full pipeline:

python3 run.py

or run specific modules:

python3 src/train_model.py
python3 src/evaluate.py
